Advanced JS (ES6): classes

L'objectif de ce TP est d'afficher un groupe de formes simples en SVG. Ces formes sont représentés (centre, forme, couleur, trajectoire ) par des instances de classes.

Les classes d'éléments géométriques et autres outils

La représentation des positions en SVG se fait selon des axes X et Y orientés depuis le coin supérieur gauche vers la droite (X) et vers le bas (Y).

Créez une classe Coordinates qui permet de stocker les coordonnées d'un point dans cette représentation et de les convertir dans un repère cartésien placé au centre de l'élément SVG.

Créez également une classe BoundingBox qui représente le rectangle englobant une forme géométrique. Son constructeur prend les coordonnées du point supérieur gauche et du coin inférieur droit. On peut alors définir un centre grâce à une méthode.

Créez enfin une classe Color dont le constructeur ne prend aucune valeur mais détermine un triplet Rouge/Vert/Bleu à passer aux formes.

Afin d'organiser le code, ces classes seront regroupées dans un module.

Les classes de formes géométriques

Ces formes pourront être des rectangles, des ellipses ou des polygones. Toutes les formes hériteront d'une même classe générique Shape.

Les formes géométriques s'inscrivent toutes dans une BoundingBox et ont une Color.

Afin d'organiser le code, les classes de formes seront regroupées dans un module. Ce module n'exportera pas la classe générique.

Les classes de trajectoires

Les formes géométriques seront animées : elles suivront des trajectoires. Les trajectoires pourront être linéaires, balistiques, élastiques ... et hériteront d'une classe générique.

Afin d'organiser le code, les classes de trajectoires seront regroupées dans un module.

Les objets animés

Créez une classe permettant de générer des objets animés. Les objets animés sont des objets qui ont pour attribut une forme géométrique et une trajectoire.

Dans le script principal, un nombre aléatoire d'objets animés seront générés et stockés dans un Set. Des boutons permettront de lancer et suspendre l'animation.
